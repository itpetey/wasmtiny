# Wasmtiny

A small WebAssembly designed specifically for the needs of the [Selium project](https://github.com/seliumlabs/selium).

It features guest suspension/resumption, metering/limiting of guest resources, and hot-pluggable shared memory.

You are most welcome to use this runtime in your projects, however please note that the project's direction will be heavily influenced by the needs of Selium.
